Projetos em ciência de dados

Desenvolvimento de um Painel Gerencial para Negócio com o Stremlit.

Neste projeto, foram aplicados conceitos de programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, juntamente com o uso de ferramentas de desenvolvimento web, como o Streamlit e o GitHub. O objetivo principal foi desenvolver um Painel Gerencial que exibisse as principais métricas de uma empresa global de marketplace de restaurantes.
O resultado final do projeto consistiu em um painel hospedado em um ambiente de nuvem e acessível por meio de um link web. Essa solução permite que o painel seja acessado de qualquer dispositivo conectado à internet.

As ferramentas utilizadas foram:

  1. Python,
  2. Jupyter Lab,
  3. Terminal,
  4. K-Means, Hierarquical Clustering, DBScan
  5. Streamlit,
  6. Streamlit Cloud, e
  7. Github
